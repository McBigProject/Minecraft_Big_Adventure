# Fonctionnalités du Mod

## Emotes Personnalisées
Vous pouvez créer vos propres emotes et les jouer.

## Synchronisation des Emotes sur le Serveur
Il n'est pas nécessaire d'envoyer les emotes personnalisées à tout le monde pour les jouer.

## Outils de Création d'Emotes Blender et Blockbench
Utilisez Blender ou Blockbench pour créer facilement vos emotes personnalisées.

## Utilisation
Par défaut, appuyez sur B pour ouvrir le menu circulaire de sélection des emotes.

## Installation des Emotes Personnalisées
Placez-les dans le dossier `.minecraft/emotes`.
(Pressez simplement sur "Ouvrir le dossier des emotes" dans l'écran de configuration du mod pour ouvrir le dossier dans votre navigateur de fichiers).
